Tag de uso Genérico

Span: inline

Um parágrafo é uma unidade de sentido que divide e organiza um texto, sendo um dos elementos mais importantes de uma redação. Para identificar um parágrafo, é possível procurar por: Recuo na margem esquerda, Início com letra maiúscula, Encerramento com ponto final.
